Transmission Main Gear Nut Wrench
This tool is used to remove the 2-1/4 inch hex nut from the main drive gear which secures the rear drive transmission pulley. To be used with 1/2 inch drive ratchet. This tool is designed to remove and install the right handed nut that holds on the transmission pulley. The inner collar is designed to be threaded onto the mainshaft. This tool is designed to stop slipping when torquing down the transmission pulley nut. See HD service book for torque specs. Remember to always apply lube to threads of tool. Use with 1-1/2 inch drive ratchet. 2007-2017 FXST
